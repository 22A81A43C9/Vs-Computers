# VS Computers Website

A modern and responsive business website for VS Computers, providing computer and printer services.

## 🚀 Features
- Responsive design (mobile + desktop)
- Clean and modern UI
- EmailJS contact form (no backend required)
- Auto-reply email system
- Form validation and user feedback
- Services section (Repair, Data Recovery, Rentals, etc.)

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- EmailJS

## 📩 Contact Functionality
Users can submit enquiries through the contact form.  
Messages are sent directly to the business email using EmailJS, and customers receive an automatic confirmation email.

## 🎯 Purpose
This project demonstrates frontend development skills and integration of third-party services to build a real-world business website without using a backend.

## 📌 Future Enhancements
- WhatsApp integration
- Backend with database (Node.js)
- Admin dashboard for managing leads
